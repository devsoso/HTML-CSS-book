# 목록
- 목록은 다른 목록 내에 중첩이 가능하다.

## 순서 있는 목록

```html
<ol>
  <li>빨간색</li>
  <li>노란색</li>
  <li>파란색</li>
</ol>
```
<ol>
  <li>빨간색</li>
  <li>노란색</li>
  <li>파란색</li>
</ol>

## 순서 없는 목록

```html
<ul>
  <li>빨간색</li>
  <li>노란색</li>
  <li>파란색</li>
</ul>
```
<ul>
  <li>빨간색</li>
  <li>노란색</li>
  <li>파란색</li>
</ul>

## 정의 목록

```html
<dl>
  <dt>빨간색</dt>
  <dd>It's Red.</dd>
  <dt>노란색</dt>
  <dd>It's Yellow.</dd>
  <dt>파란색</dt>
  <dd>It's Blue.</dd>
</dl>
```

</dl>
